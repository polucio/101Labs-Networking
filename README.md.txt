# DNS Lab

## Objective
The objective of this lab is to configure and test DNS on a Packet Tracer server.

## Steps
1. Configure IP addresses for the PC and server.
2. Set up the DNS service on the server and add an A record for `www.mypage.com`.
3. Verify the DNS resolution from the PC.

## Files
- `DNS_Lab.pkt`: Packet Tracer file for this lab.

## Lab Topology
- PC (192.168.1.5)
- Server (192.168.1.1)
- Connected via a switch.
